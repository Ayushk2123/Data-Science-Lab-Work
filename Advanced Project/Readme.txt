# Student Performance Prediction System

## Description
This project analyzes student performance and predicts final grades using machine learning models.

## Dataset
- Source: UCI / Kaggle
- Description: Student data including study time, failures, absences, and grades.

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Visualization
4. Model Building (Linear Regression & KNN)

## Results
- Study time positively impacts performance
- Failures reduce final grades
- Linear Regression performed better than KNN

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Conclusion
Student habits significantly influence academic performance.
