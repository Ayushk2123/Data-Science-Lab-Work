# Titanic Survival Analysis

## Description

This project analyzes the Titanic dataset to understand the factors that influenced passenger survival. It includes data cleaning, visualization, and a machine learning model to predict survival.

## Dataset

* Source: Public dataset (GitHub / Kaggle)
* Description: The dataset contains information about Titanic passengers such as age, gender, class, fare, and survival status.

## Steps Performed

1. Data Cleaning

   * Handled missing values in Age and Embarked columns
   * Removed unnecessary columns like Cabin, Name, and Ticket

2. Exploratory Data Analysis (EDA)

   * Analyzed survival distribution
   * Compared survival based on gender, class, and age

3. Visualization

   * Count plots for survival, gender, and class
   * Histogram for age distribution

4. Model Building

   * Used Logistic Regression
   * Split data into training and testing sets
   * Evaluated model accuracy

## Results

* Female passengers had a higher survival rate than males
* First-class passengers had better survival chances
* Age and fare influenced survival probability

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Conclusion

Passenger characteristics such as gender, class, and age played a significant role in determining survival. The model provides a basic prediction of survival based on these features.
