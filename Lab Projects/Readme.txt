# Data Science Lab Work

## Description

This repository contains all the lab assignments and projects completed as part of the Data Science course. The work includes fundamental concepts of Python programming, data analysis, and machine learning techniques.

## Lab Work

### Lab 1: Introduction to Python

* Basics of Python programming
* Variables, data types, and operators

### Lab 2: Control Statements

* Conditional statements (if-else)
* Loops (for, while)

### Lab 3: NumPy

* Array operations
* Mathematical computations using NumPy

### Lab 4: Pandas

* Data manipulation
* Data cleaning and handling datasets

### Lab 5: Regression Analysis

* Linear Regression concepts
* Model training and evaluation

### Lab 6: KNN Classification

* K-Nearest Neighbors algorithm
* Classification techniques
